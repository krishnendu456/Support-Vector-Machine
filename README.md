Iris Flower Classification using Decision Tree

This project implements a machine learning classification model using the famous Iris dataset, introduced by Ronald A. Fisher. The objective is to classify iris flowers into three species — Setosa, Versicolor, and Virginica — based on four features: sepal length, sepal width, petal length, and petal width.

A Decision Tree classifier is trained on the dataset after splitting it into training and testing sets. The model is evaluated using accuracy and classification metrics such as precision, recall, and F1-score.

This project demonstrates the fundamentals of supervised learning, train-test splitting, model training, and performance evaluation using Python and scikit-learn.
